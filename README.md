# Semantic Search Engine

https://loke-semantic-search-engine.streamlit.app/

A simple Streamlit application that demonstrates semantic search using ChromaDB and vector embeddings.

## Features

* Search by meaning, not exact keywords
* Uses ChromaDB for vector storage and retrieval
* Displays top matching documents
* Interactive Streamlit interface
* View the knowledge base before searching

## Tech Stack

* Python
* Streamlit
* ChromaDB
